# UI.md - 디자인 명세서

## 1. 디자인 철학: 글라스모피즘 (Glassmorphism)
인터페이스는 "글라스모피즘" 미학을 사용하여 현대적이고 고급스러운 느낌을 제공합니다. 이는 반투명성, 선명한 배경, 흐림 효과(Blur), 그리고 미세한 테두리를 사용하여 깊이감과 계층 구조를 만듭니다.

## 2. 색상 팔레트 & 자산

### 배경 (Background)
- **유형**: 메쉬 그라디언트(Mesh Gradient) 또는 애니메이션 CSS 그라디언트
- **색상**: 딥 블루(`4facfe`) ~ 시안(`00f2fe`) 또는 퍼플(`a18cd1`) ~ 핑크(`fbc2eb`)
- **목표**: 유리 패널이 돋보이도록 높은 대비를 제공합니다.

### 유리 패널 (컨테이너)
- **배경**: `rgba(255, 255, 255, 0.15)`
- **백드롭 필터**: `blur(12px)` 또는 `blur(16px)`
- **테두리**: `1px solid rgba(255, 255, 255, 0.3)`
- **그림자**: `0 8px 32px 0 rgba(31, 38, 135, 0.37)`
- **테두리 반경**: `16px` 또는 `24px`

### 텍스트 구현
- **기본 텍스트**: `rgba(255, 255, 255, 0.95)` (높은 가독성)
- **보조 텍스트**: `rgba(255, 255, 255, 0.7)`
- **글꼴**: 'Inter', 'Outfit', 또는 system-ui (현대적인 산세리프)

### 강조 색상 (Accent Colors)
- **주요 동작 (Call to Action)**: `rgba(255, 255, 255, 0.25)` (밝은 테두리 포함)
- **호버 상태**: 불투명도 증가 (`rgba(255, 255, 255, 0.35)`) + 약간 확대
- **활성/선택됨**: 단색 또는 고불투명도 유리

## 3. 레이아웃 구조

### A. 헤더
- 제목: "JSON Re-Formatter" (또는 유사)
- 단순하고 중앙 정렬된 큰 타이포그래피.

### B. 메인 작업 공간 (Grid/Flex)
1.  **입력 패널** (좌측/상단)
    - 파일 드롭 영역 (점선 유리 테두리)
    - 직접 붙여넣기를 위한 텍스트 영역
2.  **설정 패널** (중앙/중간)
    - 필드 목록 (드래그 가능한 칩/항목)
    - 포맷팅 토글 (스위치 스타일)
3.  **출력 패널** (우측/하단)
    - 미리보기 영역 (`pre` 태그 스타일링)
    - 액션 바: [복사] [다운로드]

## 4. 컴포넌트 스타일

### 버튼
```css
.glass-btn {
  background: rgba(255, 255, 255, 0.2);
  backdrop-filter: blur(4px);
  border: 1px solid rgba(255, 255, 255, 0.2);
  color: white;
  transition: all 0.3s ease;
}
.glass-btn:hover {
  background: rgba(255, 255, 255, 0.3);
  transform: translateY(-2px);
  box-shadow: 0 4px 15px rgba(0, 0, 0, 0.2);
}
```

### 입력 / 텍스트 영역
- 유리 테두리가 있는 투명 배경
- 가독성을 위해 엄격하게 제어된 `::placeholder` 색상
- 포커스 상태: 발광 효과 또는 더 밝은 테두리

## 5. 애니메이션
- **페이드 인**: 메인 컨테이너 초기 로드 시.
- **크기 조절 (Scale)**: 버튼 호버 시.
- **슬라이드**: 토스트 알림용 (클립보드 복사됨).
